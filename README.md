# Customer Shopping Behavior Analysis

## 📊 Project Overview

This project analyzes customer shopping behavior data to uncover purchasing patterns, customer preferences, spending trends, and actionable business insights. Using Python, MySQL, and Power BI, the project transforms raw customer transaction data into meaningful information that supports data-driven business decisions.

The analysis follows an end-to-end analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, dashboard development, and business recommendations.

---

## 🎯 Business Problem

Retail businesses generate large amounts of customer transaction data but often struggle to convert that information into actionable insights.

This project aims to answer key business questions such as:

- Which customer segments contribute the most revenue?
- What are the most popular product categories?
- How do customer demographics influence purchasing behavior?
- Which factors drive higher spending patterns?
- What opportunities exist to improve customer retention and sales performance?

---

## 🎯 Objectives

- Analyze customer purchasing behavior and spending patterns.
- Identify high-value customer segments.
- Evaluate product category performance.
- Examine demographic trends across customer groups.
- Generate actionable business insights from transactional data.
- Build an interactive dashboard for business decision-making.

## 📁 Dataset Description

The dataset contains customer shopping behavior information, including demographic characteristics, purchasing activity, and transaction details.

### Key Fields

| Field | Description |
|---------|-------------|
| Customer ID | Unique customer identifier |
| Age | Customer age |
| Gender | Customer gender |
| Category | Product category purchased |
| Purchase Amount | Transaction value |
| Payment Method | Customer payment method |
| Subscription Status | Customer subscription information |
| Previous Purchases | Historical purchasing activity |
| Frequency of Purchases | Purchase frequency indicator |
| Season | Purchase season |
